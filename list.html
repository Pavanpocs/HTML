Definition List:
It is also called asDescriptive List.
dl tag are use for to specify the definition to definition term
dl and dd tag are child tags of DD(Parent Tag) tag.
dl use for define the main term of items.
dd used for definition of main term in items.

Example: Definition List tag are dl : to specify the definition term.
dt : dt tag are used for to specify the definition data.
dd : dd tag are used to specify the definition of definition term.
